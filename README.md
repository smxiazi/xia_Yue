# xia_Yue (瞎越)

* burp插件
* 主要用于测试越权、未授权
* 感谢名单：Moonlit

### 获取方式
* 目前还在测试，想要的可以微信找我要，记得点可🌟在来问我要。

### 注意
* 默认使用jdk1.8编译
* 在最新版的burp2.x中jdk为1x,会导致插件不可用,请下载jdk16版本试试,若还不行，请自行下载源码使用当前电脑的jdk1x进行编译,谢谢。

### 插件描述
* 返回 ✔️ 表示大小和原始数据大小一致
* 返回（数字）表示和原始数据包相差的大小
